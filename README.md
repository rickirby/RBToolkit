# RB_Toolkit_Core_iOS
Often used extension of UIKit


### How to use:
on Cartfile, add following lines:

git "git@bitbucket.org:rb-scando/rb_toolkit_core_ios.git" == *version-number*

Then "carthage update --use-ssh"
